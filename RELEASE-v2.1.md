## 🎯 核心功能

### 1. 威胁检测引擎 🔍
- **200 个威胁特征** - 覆盖 9 大类攻击模式
- **实时扫描** - 文件/目录扫描
- **风险评级** - critical/high/medium/low

### 2. 静态代码分析 🔬
- **AST 解析** - 深度代码分析
- **敏感 API 检测** - 25+ 危险 API
- **VirusTotal 集成** - 云端威胁情报

### 3. 安装风险评估 ⚠️
- **4 级风险评分** - 0-100 分
- **自动拦截** - 高风险阻止
- **详细报告** - 完整风险因素

### 4. 运行时拦截 🛡️
- **文件访问监控** - 16 个敏感路径
- **实时监控** - Linux inotify
- **网络请求拦截** - 主机黑白名单
- **hosts 文件拦截** - 系统级拦截
- **进程级追踪** - psutil 集成

### 5. 安全日志中心 📋
- **统一日志** - 所有安全事件
- **查询导出** - JSON/CSV
- **摘要统计** - 24h/7d/30d

### 6. WebSocket 推送 📡
- **实时通知** - 安全事件推送
- **客户端管理** - 多客户端支持

### 7. CLI 工具 🛠️
- **check** - 技能风险检查
- **scan** - 目录扫描
- **status** - 系统状态
- **logs** - 日志管理

## 📦 安装方式

### ClawHub 安装
```bash
clawhub install claw-def
```

### 手动安装
```bash
git clone https://github.com/clawdef/claw-def.git
cd claw-def
git checkout v2.1
```

---

## 🚀 快速开始

### 检查技能风险
```bash
python3 src/cli.py check ./target-skill
```

### 扫描目录
```bash
python3 src/cli.py scan ./suspicious-dir
```

### 查看系统状态
```bash
python3 src/cli.py status
```
