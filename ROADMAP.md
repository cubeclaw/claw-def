## 🤝 如何贡献

### 提交特征
```bash
# 格式
{
  "id": "XX001",
  "name": "特征名称",
  "pattern": "正则表达式",
  "severity": "critical|high|medium|low",
  "description": "描述"
}
```

### 报告误报
- Issue: https://github.com/clawdef/claw-def/issues
- 标签：`false-positive`

### 提交样本
- 恶意样本：请通过 Issue 私下提交
- 正常样本：PR 到 `tests/samples/`
